### 🚀 Quick Start

#### Prerequisites
- Node.js 20+ and npm


#### Installation

1. **Clone the repository**
```bash
git clone https://github.com/galafis/ibm-js-fullstack-guestbook-app.git
cd ibm-js-fullstack-guestbook-app
```

2. **Install dependencies**
```bash
npm install
```

#### Running

```bash
npm run dev
```
